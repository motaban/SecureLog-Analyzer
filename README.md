Diseñé e implementé un dashboard web integral para el análisis y visualización de logs de seguridad, permitiendo a los equipos de SOC identificar, analizar y responder a amenazas cibernéticas de manera proactiva y eficiente.

Arquitectura Full-Stack: Desarrollé una solución completa utilizando React (TypeScript) para una interfaz dinámica y responsive con tema oscuro, y Node.js con APIs RESTful seguras en el backend. Utilicé MySQL para el almacenamiento persistente de datos, implementando encriptación AES-256 para información sensible.

Análisis y Automatización: Integré scripts en Python (Pandas, Scikit-learn) para el procesamiento automatizado de logs y la detección básica de anomalías, clasificando amenazas como DDoS, malware o accesos no autorizados y asignando scores de riesgo.

Visualización de Datos en Tiempo Real: Implementé WebSockets (Socket.io) para proporcionar alertas y actualizaciones en vivo, junto con gráficos interactivos, mapas de calor y herramientas de filtrado avanzado (por IP, fecha, tipo de amenaza).

Robustez y Seguridad: Aplicando conocimientos de certificaciones en ciberseguridad, fortifiqué la aplicación con autenticación multifactor (MFA), validación estricta de entradas para prevenir inyecciones (XSS/SQL), y políticas de least privilege. Integré APIs de threat intelligence como VirusTotal para enriquecer el análisis.

Funcionalidades Clave: El sistema incluye un módulo de simulación de ataques para pruebas, generación de informes accionables (exportables a PDF/CSV), y un completo logging de auditoría para el rastreo de todos los cambios realizados.

Tecnologías: React, TypeScript, Node.js, Python, MySQL, PHP, Docker, Socket.io, Pandas, Git.
