Dashboard de Análisis de Logs de Seguridad

Una solución full-stack para monitorizar, analizar y visualizar logs de seguridad en tiempo real, diseñada para entornos SOC junior con énfasis en escalabilidad y conformidad con estándares NIST.

Ver Demo Código en GitHub

Características Principales

Interfaz Intuitiva

Dashboard con tema oscuro (grises y negros) y diseño responsive, adaptado para dispositivos móviles y desktop.

Ingreso y Visualización

Procesamiento de logs en 4 pasos: carga, filtrado por tipo de amenaza, análisis automatizado y generación de informes.

Autenticación MFA

Autenticación multifactor con validación de entradas para prevenir inyecciones XSS y SQL, incluyendo sanitización de datos.

Almacenamiento Híbrido

MySQL para datos persistentes y LocalStorage para sesiones temporales, con encriptación AES para logs sensibles.

Visualizaciones Interactivas

Gráficos de tendencias de amenazas, mapas de calor para eventos sospechosos y alertas en tiempo real vía WebSockets.

Filtrado Avanzado

Filtrado por fecha, IP, tipo de amenaza (DDoS, malware) y prioridad, con opciones de exportación a PDF/CSV.

Enfoque en Ciberseguridad

  • Validación estricta de entradas y salidas para mitigar riesgos como XSS, CSRF y SQL Injection
  • Encriptación AES-256 para datos en reposo y en tránsito (HTTPS)
  • Análisis de amenazas automatizado con detección de patrones sospechosos
  • Cumplimiento con principios de least privilege y logging detallado para auditorías
  • Integración con APIs externas (VirusTotal) con manejo seguro de tokens
  • Pruebas de vulnerabilidades con OWASP ZAP en el pipeline de CI/CD
